1 Components
* Function Component
```javascript
import React from 'react';
const Header = () => {
    return(
        <div>Header</div>
    );
};
export default Header;
```
* Class Component
```javascript
import React, { Component } from 'react';
class Header extends Component{
    render() {
        return(
            <div>Header</div>
        );
    }
}
export default Header;
```
